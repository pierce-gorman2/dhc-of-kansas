# DHC Kansas Website Starter

This is a public-facing static site for a Dave's Hot Chicken franchise company operating in Kansas.

## Current focus
- full-screen hero landing section
- community involvement
- news and updates
- central locations page
- about page
- podcast link
- separate Wichita, Manhattan, Lawrence, and Topeka pages
- separate detail pages for community and news sections

## Files
- `index.html` -> home page
- `about.html` -> about page
- `community.html` -> community overview page
- `news.html` -> news overview page
- `locations.html` -> grouped locations overview page
- `wichita.html` -> Wichita location page
- `manhattan.html` -> Manhattan location page
- `lawrence.html` -> Lawrence location page
- `topeka.html` -> Topeka location page
- `schools.html` -> school partnerships page
- `community-partners.html` -> community partners page
- `location-progress.html` -> location progress page
- `company-media.html` -> company media page
- `styles.css` -> styling
- `script.js` -> mobile menu and reveal effects
